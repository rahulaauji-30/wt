Data Types

In Oracle Database, data types define the type of data that can be stored in a column of a table or a variable. Each column in a table must be assigned a specific data type, which helps Oracle manage the storage and retrieval of data efficiently. Here's an overview of some commonly used data types in Oracle DBMS.

  1. Numeric Data Types
    • NUMBER: Used for storing fixed-point or floating-point numbers. It can have precision and scale. 
                          
                        salary NUMBER(10, 2);
  2. Character Data Types
    • CHAR:Fixed-length character data. Trailing spaces are padded. 
                          
                        first_name CHAR(50);
    • VARCHAR2:Variable-length character data. Trailing spaces are not padded. 
                              
                            last_name VARCHAR2(50);
  3. Date and Time Data Types
    • DATE:Stores date and time values. 
                          
                        hire_date DATE;
    • TIMESTAMP:Stores date and time down to fractional seconds. 
                              
                            last_updated TIMESTAMP;
  4. Large Object Data Types
    • CLOB (Character Large Object):Stores large amounts of character data. 
                              
                            description CLOB;
    • BLOB (Binary Large Object):Stores binary data like images or documents. 
                              
                            document BLOB;
  5. Boolean Data Type
    • BOOLEAN:Represents binary (true/false) values. Introduced in Oracle 12c. 
                              
                            is_active BOOLEAN;
  6. Row Identifier Data Types
    • ROWID:Represents the unique address of a row in a table. It is often used internally by Oracle. 
                              
                            row_id ROWID;
  7. Interval Data Type
    • INTERVAL YEAR TO MONTH:Represents a period of time in years and months. 
                              
                            service_length INTERVAL YEAR TO MONTH;
  8. XML Data Type
    • XMLTYPE:Stores XML data. 
                              
                            xml_data XMLTYPE;
  9. Geospatial Data Types
    • Oracle supports geospatial data types for handling spatial data.
  10. User-Defined Data Types
    • OBJECT Type:Allows you to define your own data types using the CREATE TYPE statement. 
                              
                            CREATE TYPE rahul;
    • VARRAY (Variable-Size Array):An array with a variable size.
    • Nested Table:A table with a varying number of rows.
Examples 
        
            CREATE TABLE employees (
                employee_id NUMBER(5) PRIMARY KEY,
                first_name VARCHAR2(50),
                last_name VARCHAR2(50),
                hire_date DATE,
                salary NUMBER(10, 2),
                notes CLOB
            );
