Getting Started

Basic SQL Commands

Begin your exploration with fundamental SQL commands for database interaction

                    
                    -- Create a table
                    CREATE TABLE employees (
                        employee_id NUMBER PRIMARY KEY,
                        first_name VARCHAR2(50),
                        last_name VARCHAR2(50),
                        hire_date DATE,
                        salary NUMBER
                    );
                    
                    
                    --Insert data into the table
                    INSERT INTO employees (employee_id, first_name, last_name, hire_date, salary)
                    VALUES (1, 'John', 'Doe', TO_DATE('2024-01-01', 'YYYY-MM-DD'), 50000);
                    
                    
                    -- Query data
                    SELECT * FROM employees;
